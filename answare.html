Question-1: What are the different ways to select an element in the DOM?

In the DOM, we can select an element through different ways. In this case, the ways depend on the situation. Whatever, let's try to see some selectors in the DOM by which we will be able to select an element. The most well known selectors in DOM are listed below: querySelector(), querySelectorAll(), getElementById(), getElementsByClassName() and so on. querySelector(), querySelectorAll() can be used to select an element in css styling. But there is a little difference between querySelector() and querySelectorAll(), that is: querySelector() is able to select one element through either id or class whereas querySelectorAll() is able to select one or more than one elements through either id or class and produce a NodeList of the given elements. We can use getElementById() to select an individual element with respect to only specific id of the given element. In the same way, getElementsByClassName() is used to select multiple elements with respect to the class name and it can produce a HTMLCollection of the given selected elements.

Question-2: What is the difference between innerHtml, innerText and textContent?

The innerHTML is a property that can return the text content of the element, including all spacing and inner HTML tags. The innerText is also a property that is able to return just the text content of the element and all its children. The textContent property also ignores all HTML tags and returns only the text.

Question-3: What is event delegation in the DOM?

We can consider the event delegation as a way or technique in which a parent element handles events for its child elements, even if the children are added dynamically after the page loads.

Question-4: What is event bubbling in the DOM?

In this case, when we click a button, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event is bubbled up to the elements' parent.

Question-5: How do you create, add and remove element using javascript?

We can create a new element in the DOM of javascript using createElement() method. In the same way, the created element can be added inside the parrent element using appendChild() method. in this case, if we would like to remove the given element after adding, we can use removeChild() method.
